# shortcuts

## navigation
```html
SHIFT + SHIFT - search all files and classes
CTRL + N       - search classes
CTRL + SHIFT + F - search text in all files
CTRL + SHIFT + R - replace text in all files
CTRL + F - search text in current file
CTRL + R - replace text in current file

CTRL + click - navigate (on field, on method, on class)
ALT + SHIFT + < - go back
ALT + SHIFT + > - go forward
```

## refactor
```html
SHIFT + F6      - refactor
CTRL + ALT + M  - extract method
CTRL + ALT + F  - extract field
CTRL + ALT + P  - extract parameters
CTRL + ALT + O  - optimize imports
CTRL + ALT + L  - reformat code

extract interface: right click refactor
```

## code
```html
CTRL + D - duplicate
CTRL + SHIFT up - move code up
CTRL + SHIFT down - move code down
generate test: click on class name, ALT + ENTER
```
